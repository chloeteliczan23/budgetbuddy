# BudgetBuddy

BudgetBuddy is a modern, student‑focused budgeting website designed to help college students build financial confidence through simple tools, clear guidance, and accessible financial education. The site includes bilingual support (EN/ES), dark/light mode, interactive budgeting resources, and a clean, responsive UI built with Tailwind CSS.

This project demonstrates full‑cycle web development: planning, building, refining, deploying, and analyzing a complete multi‑page application.

## ✨ Features

### 🌙 Dark/Light Mode  
- Implemented using Tailwind’s `darkMode: 'class'`  
- Toggle button in the header  
- Saves user preference in `localStorage`  

### 🌎 English / Spanish Translation  
- Custom translation system using `lang.js`  
- All text elements use `data-key` attributes  
- Language preference saved in `localStorage`  

### 📄 Multi‑Page Layout  
- Home  
- Budgeting Tools  
- Financial Education  
- Resources  
- Contact  

### 📚 Financial Education Modules  
- Budgeting Basics (links to Consumer.gov)  
- Credit & Debt 101 (YouTube playlist)  
- Student Loans (links to U.S. Department of Education)  

### 📥 Downloadable Resources  
- Monthly Budget Template (Excel)  
- Weekly Expense Log (PDF)  

### 📱 Responsive Design  
- Tailwind CSS ensures clean layout across devices

## 🚀 Deployment

The site is deployed using **GitHub Pages**.

### Steps:
1. Create a GitHub repository  
2. Upload all project files  
3. Go to **Settings → Pages**  
4. Select `main` branch  
5. GitHub generates a live URL
